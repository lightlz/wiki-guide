# wiki 翻译指南

先认真看这个文档，有不明白的再问。

## 工作内容

翻译 + 校对文档（英译中）

## 待遇

1000 中文字 / 100 RMB

_组长有兴趣另谈。_

## 工作流程

- 查看英文原文文档

- 认领自己想翻译的章节（和负责人联系并说明完成时间）

- 翻译完成相互校对

- 上传至 github

## 相关工具

- markdown 编辑器

	翻译的文本使用 markdown 格式编写

- github 

    翻译好的文档需要上传至 github
